你永远会低估你一周可以做的事情，在这里记录一下我这周看到的好东西。有些链接可能需要科学上网。

在美国的同学估计都了解到了HR1044法案通过的众议院的新闻，国内的同学可以把这个事情当个故事看。这个主要是和外国人在美国拿绿卡(永久居住权)有关。

一些背景资料，目前绿卡每年发放是限制人数，每个国家都是有配额的。目前只有印度和中国需要排队，印度人大概需要等10年绿卡，我国大概现在需要3-4年。美国一项法案的通过需要先通过众议院(根据州的人数分配的代表)，参议院(每个州2个人)，以及总统签字。

这个HR1044法案的名称是高技能移民公平方案，该提案提议取消职业移民绿卡的国别限制，从而可能消灭目前排期过长的问题。里面有一些具体的规则，简而言之，最后的效果就是可以让印度移民拿到全部绿卡名额。中国人可能在4~5年完全拿不到任何绿卡，排期时间大大延长。具体细节中会对部分已经在排队的中国移民有利，对还未排期的除印度以外的其他族裔都有很大的坏处。因为大家都必须要等消化完印度10年排期的人，他们的优先级比较高。

本文记录了自己在root已停止支持的红米手机以及破解加密微信聊天记录数据库的一些心得和体会。

root 手机

Root小米手机分为以下几步：

  1. 刷miui的开发者版本
  2. 解锁小米手机
  3. 开启root权限

我手里有一部红米3的手机，而小米官方对于这款手机已经停止了维护，所以在miui官网上已经找不到相应的卡刷包。解决办法就是上论坛下以往的，我找到的在这里。

我先下载了卡刷包，之后把zip包直接放到sd卡里。在系统更新页面点击右上角，选择使用本地文件进行更新。注意这里会抹掉所有手机的应用，在漫长的等待之后就把自己miui版本变成了开发版。

接下来首先打开开发者模式，运许usb调试，打开fastboot刷机模式。另外在授权管理里面找root授权管理，把自己的手机与mi account绑定，准备解锁。

bit操作题目思考

说实话这类题目比较考验智商，如果之前没有见过的话做出来的难度比较大。基本思路很多都是利用XOR解题

find bit parity

这题就是找奇偶校验位的值，有奇数个1就是1，有偶数个1就是0。在513 datalab出现过，现在在思考感觉没那么难了。基本思路就是把所有的1进行XOR，看最后是1还是0.

x ^= x >> 16 
x ^= x >> 8 
x ^= x >> 4 
x ^= x >> 2 
x ^= x >> 1 
x &= 1;

找出2个只出现一次的数值

这个题目可以全部XOR，得到x XOR y。之后取其中一位，然后把数组分成2部分，一半有这位，一半没有，然后用前两题的思路。
