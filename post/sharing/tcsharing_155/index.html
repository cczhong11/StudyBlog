新闻思考

这周东方甄选和董宇辉的事件引发了很大的争议事件起源于董宇辉在直播间与东方甄选的编辑（“小编”）之间的不愉快。小编声称，董宇辉的直播稿件是团队共同努力的结果，这与之前董宇辉作为独立创作者的形象相矛盾，引发了他的粉丝不满。

随后，东方甄选（新东方的子公司）的CEO孙东旭对董宇辉做出了贬低性评论，暗示他只是一个普通员工，并不值得特殊对待。这导致了董宇辉粉丝的强烈反噬。因为这场争议和粉丝的反弹，新东方决定解雇东方甄选的CEO孙东旭

在我国的职场下，大多数情况是老板说了算，员工只能服从。但是在东方甄选和董宇辉中，董宇辉对于公司实在是太重要了，他的粉丝数量和影响力都是公司的宝贵资源。在这种情况下，CEO还是PUA了董宇辉，这让粉丝们很不满，强烈的共情了打工人的心态。毫无疑问在新时代的发展下，尤其是直播平台，头部主播的效应非常明显，观众就是冲着主播来的，主播影响力非常大。如果还是以前的上下级关系，头部主播得不到公司足够的尊重，那么很容易就会离开，这对于公司来说是非常大的损失。所以应该要转变一下思维，公司和主播是一种合作关系，而不是上下级关系。这样才能让公司和主播都能够获得更大的价值。

我在现在的公司里体会到的一点就是老板是support员工而存在的，因为干活的还是员工，老板只是提供资源和支持。这样的关系才能让公司和员工都能够获得更大的价值。希望在未来的中国职场中，这种关系能够越来越多。

【1】Nature公布2023年度十大科学人物 ChatGPT上榜 顶级科学期刊《Nature》2023 年在科学领域产生重大影响的十位人物。 让人惊讶的是，今年还有一位非人类入选 ，那就是过去一年出尽风头的聊天机器人： ChatGPT。 《自然》特写部主编 Richard Monastersky 表示： “虽然这个工具不算人物，也不完全满足《自然》十大人物的评选条件，但我们将其破例纳入榜单，从而承认生成式 AI 给科学发展和进步带来的巨大改变。” 其中OpenAI的首席科学家 Ilya Sutskever也上榜了…

【2】美国商务部长雷蒙多周一在访问新罕布什尔州纳舒厄时接受彭博社采访透露，美国正在研究英伟达为中国开发的三款新型芯片的具体细节，“我们会检查每个新芯片的每一个规格，显然是为了确保它不违反出口管制。我们定期与英伟达交流，他们是很好的合作伙伴，我们与他们有着密切的工作关系。他们和我们共享信息”。美国商务部本月早些时候曾誓言限制任何为中国提供人工智能能力的新芯片。

【5】根据 The Verge 的报道，字节跳动被OpenAI封杀！

因为字节也违反了 OpenAI 的服务条款，一直在使用GPT生成的数据在中国训练自己的竞争模型，从而违反了微软和OpenAI的开发人员许可证。字节跳动通过微软获取 OpenAI 的访问权限。

字节跳动的内部文件显示，他们在开发名为 Project Seed 的基础 LLM 时，几乎在每个阶段都依赖 OpenAI 的 API，包括用于训练和评估模型。由于过度依赖 OpenAI 的 API，Project Seed 的员工经常达到了 API 访问的最大限额。

涉及的员工清楚这一行为的含义，甚至讨论了如何通过“数据脱敏”来“洗白”证据。这种滥用如此普遍，以至于 Project Seed 的员工经常达到 API 访问的最大配额。

字节跳动回应称，GPT生成的数据已经在今年年中左右，从字节跳动的训练数据中删除。

【8】#字节跳动再回应OpenAI封禁账户#：正与OpenAI联系沟通

据界面新闻 近日，有外媒报道称，字节跳动在使用OpenAI技术开发自己的大语言模型，违反了OpenAI服务条款。对此，字节跳动相关负责人回应称，公司在使用OpenAI相关服务时，强调要遵守其使用条款。我们也正与OpenAI联系沟通，以澄清外部报道可能引发的误解。

以下是公司使用OpenAI服务相关情况的介绍：

1、今年年初，当技术团队刚开始进行大模型的初期探索时，有部分工程师将GPT的API服务应用于较小模型的实验性项目研究中。该模型仅为测试，没有计划上线，也从未对外使用。在4月公司引入GPT API调用规范检查后，这种做法已经停止。

2、早在今年4月，字节大模型团队已经提出了明确的内部要求，不得将GPT模型生成的数据添加到字节大模型的训练数据集，并培训工程师团队在使用GPT时遵守服务条款。

3、9月，公司内部又进行了一轮检查，采取措施进一步保证对GPT的API 调用符合规范要求。例如分批次抽样检测模型输出结果与GPT的相似度，避免数据标注人员私自使用GPT。

4、未来几天里，我们会再次全面检查，以确保严格遵守相关服务的使用条款。

【9】社会：日本46%的二十多岁男子没谈过恋爱，不结婚的最大原因是没钱

据日本婚姻调查企业「リクルート ブライダル総研」于12日公布的调查数据显示，日本20～29岁未婚男性中，至今从未与异性交往过的比例升至46.0%，创历史新高。相比上次调查（2021年）的34.2%大幅上升约12个百分点。30～39岁为41.2%（上回35.4%），40～49岁为22.9%（上回27.1%）。而现在有恋人的比例分别为20～29岁26.7%，30～39岁27.6%，40～49岁14.4%。对于未来的结婚意向，回答想要结婚的男性比例为43.5%（上回47.1%），不想结婚的理由中选择“会导致经济不宽裕”的人最多，占42.5%。

同时，女性也存在和男性相同的趋势，20～29岁未婚女性中，至今从未与异性交往过的比例为29.8%（上回24.3%），30～39岁为25.3%（上回23.0%），40～49岁为26.5%（上回19.9%）。现在有恋人的比例分别为20～29岁43.2%，30～39岁31.1%，40～49岁27.7%。回答想要结婚的女性比例为49.3%（上回59.7%），不想结婚的理由中选择“行动和生活方式会受到限制”的人最多，占40.5%。

【10】著名语言学习App Duolingo（多邻国） 本周发布《2023 全球语言学习趋势报告》，总结全球语言学习排行榜，该报告基于全球超过 5 亿多邻国学习者的数据，分析总结了最新的语言学习趋势和学习者的行为习惯：

  1. 英语毫无悬念仍是全球学习榜上最受欢迎的语言。 今年英语在 122 个国家和地区荣登人气榜首，比去年多出 3 个国家（阿富汗、马尔代夫和马他耳）。在世界很多地方，个人英语能力对取得经济成功和学业成就都至关重要，而亚洲用户尤其专注于通过学习英语来支持他们的教育。从数据上看，大部分英语学习者（印度：40%；越南：50%；中国：54%) 都把教育作为语言学习的主要动机。值得一提的是，在多邻国用户最多的地区中，印度用户每天完成的英语课程量最多。

  2. 韩语升至全球第六。 2023 年，韩语在全球语言排名中超越意大利语，升至第 6 位。 韩语跻身全球数个大国的人气排名前十的语言，其中印度的韩语学习人数同比增长了 75%。
