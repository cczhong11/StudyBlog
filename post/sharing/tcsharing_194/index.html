新闻思考

这周openai发布了新模型gpt-o1,这个模型推理能力得到了很大的提升，在国际数学奥林匹克（IMO）的资格考试中，GPT-4o 仅正确解决了 13% 的问题，而推理模型的得分率则高达 83%。他们的编码能力在 Codeforces 竞赛中达到了top 89%。这个真的已经非常厉害了，比大多数人在竞赛中获得的成绩都要好。

但是对于我来说这个新模型可能并不是在所有场景都适用。新模型的输出结果非常好，但有的时候可能太多了。

比如我想让模型写一段profile import time的代码。它经过了如下的思考

Breaking down the task

I’m thinking through how to identify imported packages and measure their load times in Python, considering methods like ‘python -v’, the ‘trace’ module, or custom import hooks.

Profiling imports

I’m thinking through how to profile Python module imports by using the ‘-v’ option. This reveals details about modules and their sources when scripting.

Tracing and timing imports

Using Python’s ‘trace’ module, I’m mapping program execution and including imports. Additionally, I’m spotlighting a custom import hook to measure explicit import times.

gpt-o1一下子输出了8种方法。也可以帮助我了解了python各种库的不同的用法。但是相对来说输出的时间就比较长了，思考花了7秒钟。如果我只想用一种方法的话就有点杀鸡用牛刀了。

经过几天的使用，我发现我问的大多数问题其实gpt-4o模型就足够了，用gpt-o1在大多数情况下没有必要。但是我也可以通过和gpt-o1的交互来学习到更多的知识。

【3】OpenAI未来或不会推出GPT5 OpenAI发布新模型

北京时间凌晨一点，OpenAI 突然进行了重大更新。已经预热了接近一年的 Q*/草莓项目，传说中能够进行高级推理的大语言模型，今晚终于露出了真面目。OpenAI 发推表示，刚刚推出了 OpenAI o1-preview 模型——名字的含义之前外媒已经爆料过，o1 应该代表着 OpenAI 下一代大模型的 Orion（猎户座）一代。OpenAI 在发布文档中写道，新模型在推理能力上代表了人工智能能力的新水平，因此，计数器将重置为 1 ——这意味着未来很可能不会有 GPT-5 了，o1 将代表未来 OpenAI 的最强水平。且从今天开始，ChatGPTPlus 和 Team 用户就能够直接访问模型。（极客公园）

【4】国家网信办发布《人工智能生成合成内容标识办法（征求意见稿）》

国家互联网信息办公室发布《人工智能生成合成内容标识办法（征求意见稿）》。其中提到，人工智能生成合成内容标识包括显式标识和隐式标识。显式标识是指在生成合成内容或者交互场景界面中添加的，以文字、声音、图形等方式呈现并可被用户明显感知到的标识。隐式标识是指采取技术措施在生成合成内容文件数据中添加的，不易被用户明显感知到的标识。服务提供者提供的生成合成服务属于《互联网信息服务深度合成管理规定》第十七条第一款情形的，应当按照有关要求对生成合成内容添加显式标识。
