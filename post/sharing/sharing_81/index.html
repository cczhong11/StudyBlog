国内新闻

【1】全国多地开启拉闸限电模式 由于电力供应不足，浙江义乌、金华、宁波、温州，湖南长沙、江西等多地陆续发布有关“双控”、“有序用电”的通知。浙江部分工厂已收到“停电限产”通知。一边是经济复苏，工业用电猛增，一边是煤价提涨、电煤供应紧张，发改委紧急协调限制煤价，火电厂压力山大，再加上“节能降耗”指标压力，多年不见的“拉闸限电”再现。

湖南：限电措施旨在保障全省电网平稳运行和电力可靠供应 本月8日，湖南省发改委发出《关于启动2020年全省迎峰渡冬有序用电的紧急通知》，指出湖南省今年“入冬早、降温快”，电力负荷快速攀升。通告指出“全省最大负荷已达3093万千瓦，超过冬季历史记录，日最大用电量6.06亿千瓦时，同比增长14.1%”。为保障全省电网平稳运行和电力可靠供应，发改委宣布即日起全省启动“有序用电”，要求各市州及企业严格执行相关预案。通告提到，“有序用电”时段为每日上午十时半至正午、下午四时半至晚上八时半。通告强调在措施生效期间，将优先保障居民生活、关键公共设施和重点企业用电，而适当压限行政单位及景观用电。然而据传媒日前报导，“有序用电”措施生效以来，长沙市有市民报称因写字楼停电，而需要走20、30层楼梯上班，其余正常生活秩序也受到严重影响，引发网络舆论热议。国家电网湖南省电力新闻发言人陈浩解释，近年湖南全省工业、商业、居民生活用电负荷及用电量持续上升，但电源装机容量未有大幅增加，发电能力相对滞后。国家电网湖南省电力副总经理张孝军则指出，湖南省现有电源装机容量在今年冬季可能无法实现满负荷发电。

【浙江：措施基于“双控”及“减煤”精神】在浙江省，机关事务管理局日前发出通告，要求省级单位办公区域在气温降至摄氏3度或以下时，才能开启空调（暖气）等取暖设备，且设备设定温度不得高于摄氏16度，借以节能减碳。机关事务管理局公共机构节能处表明，上述规定的依据，包括本月11日“全省能源‘双控’和‘减煤’攻坚电视电话会议”之精神，以及省领导的具体要求。浙江省委、省政府新闻网站《浙江在线》报导提到，“双控”指控制能源消耗总量和强度，而“减煤”指削减煤炭消费总量。

【浙江多地限电 家里空调都不能开了】近日，南方多地出现“拉闸限电”的现象，引起了广泛关注。据了解，义乌部分城区为限电关闭了路灯、还有部分写字楼也关闭了电梯，而且该措施会持续到年底。有网友称这两天走在路上都黑漆漆的看不清路，家里也不能开空调。

【2】上海二手房成交量爆发 达四年高峰

上海市二手房市场11月表现强势，二手住宅成交套数约为3.2万套，环比增长13.5%，同比增长81.4%，11月成交量创4年来新高。易居研究院智库中心研究总监严跃进分析说，上海今年的楼市表现比较好，二手房交易、新房交易、土地交易以及豪宅交易都比较好，四个指数不错的表现都说明上海已经走出了一个历史的低谷。另参考搜狐焦点11月的上海楼市报告，上海各区域房价可谓是天差地别，区域间最大价差超12万。现阶段市区内黄浦区以146553元/平方米的均价占据首位，二、三名分别为卢湾区145562元/平方米和静安区107760元/平方米，而郊区金山目前均价仅为23565元/平方米。

【3】马云湖畔大学云南分校因规划调整停止招标由原阿里巴巴集团董事局主席马云等企业家和学者发起创办的湖畔大学此前确定在云南昆明设立分校，但最新消息称，该分校因规划调整停止招标。据报道，昆明市西山区滇池西岸片区征地拆迁工作指挥部综合协调处在12月10日发布公告，因片区控规编制范围调整，现终止西山区白鱼口片区控制性详细规划项目。

【4】中央经济工作会议：强化反垄断和防止资本无序扩张

中央经济工作会议12月16日至18日在北京举行。会议指出，强化反垄断和防止资本无序扩张。反垄断、反不正当竞争，是完善社会主义市场经济体制、推动高质量发展的内在要求。国家支持平台企业创新发展、增强国际竞争力，支持公有制经济和非公有制经济共同发展，同时要依法规范发展，健全数字规则。要完善平台企业垄断认定、数据收集使用管理、消费者权益保护等方面的法律规范。要加强规制，提升监管能力，坚决反对垄断和不正当竞争行为。金融创新必须在审慎监管的前提下进行。（新华社）

【5】中央经济工作会议：解决好大城市住房突出问题

中央经济工作会议12月16日至18日在北京举行。会议指出，解决好大城市住房突出问题。住房问题关系民生福祉。要坚持房子是用来住的、不是用来炒的定位，因地制宜、多策并举，促进房地产市场平稳健康发展。要高度重视保障性租赁住房建设，加快完善长租房政策，逐步使租购住房在享受公共服务上具有同等权利，规范发展长租房市场。土地供应要向租赁住房建设倾斜，单列租赁住房用地计划，探索利用集体建设用地和企事业单位自有闲置土地建设租赁住房，国有和民营企业都要发挥功能作用。要降低租赁住房税费负担，整顿租赁市场秩序，规范市场行为，对租金水平进行合理调控。

【6】“嫦娥五号”返回器在内蒙古成功着陆

周四凌晨1点59分，“嫦娥五号”返回器已在内蒙古预定区域成功着陆，标志中国首次完成从外天体采样返回任务。航天专家指，此次任务实现了中国航天史上多项重大突破，包括首次月面采样、月面起飞、月球轨道交会对接、带样返回等，完成了中国探月工程“绕、落、回”三步走发展规划。国家主席习近平发送贺电，形容中国航天向前迈出一大步，将为深化人类对月球成因和太阳系演化历史的科学认知作出贡献。根据任务目标，“嫦娥五号”此次带回2公斤月球样本，包括1.5公斤月表样本和0.5公斤钻取样本。人类上一次从月球带回样本是在1976年，苏联“月球24号”运回约170克样本。在那之前，1969年至1972年间，美国“阿波罗计划”数次登月的宇航员从月球采集了将近400公斤岩石和土壤，这些样本随载人飞船运回地球。

【2】新冠疫苗去哪接种

国务院联防联控机制举行新闻发布会通报：①目前各地都在积极落实为群众接种疫苗提供便利服务的各项措施，各地将按要求及时公布接种点和接种时间，大家可关注所在地的卫生行政部门或疾病预防控制机构发布的信息。②也会根据接种需求考虑设立临时接种点，上门服务。③接种条件将有严格规范。④所有接种点都会提供接种新冠疫苗的凭证。提醒：接种疫苗要携带身份证，做好疫苗登记！

【国家卫健委：这些重点人群优先接种新冠疫苗】国务院联防联控机制举行新闻发布会介绍：冬春季节到来之际，对部分重点人群开展新冠疫苗接种工作，对于疫情防控具有重要的意义。比如冷链物流人员、海关边检人员、医疗疾控人员，以及农贸市场、海鲜市场的工作人员，还有公共交通的工作人员等，他们由于工作的关系，被新冠病毒感染的风险相对比其他人群更高，对这部分重点人群的接种，是我国新冠疫苗接种“分步走”第一步。后续随着疫苗附条件上市或者正式批准上市，特别是随着生产供应保障能力、产量逐渐提升，对于涉及数亿人的老年人和有基础疾病的相对感染以后危险性较大的高危人群，以及后续的普通人群，疫苗接种工作都会全面有序地铺开。

【3】中国驻美使馆调整回国检测标准

据中国驻美使馆，近期从美输入确诊病例大量增加，经查，多数确诊病例与旅途中转感染有关，部分也与中介机构、实验室存在违规操作以及检测后不能遵守居家隔离要求有关。驻美国使领馆对确有紧急旅行需要的乘客进行新冠病毒核酸检测、血清抗体检测相关要求作出进一步规范和说明：检测报告必须同时明示血清（Serum）、IgM、静脉血（Venous）等字样，不再接受指尖血方式完成的检测；核酸检测报告除须明示核酸（NAAT）、rt-PCR等字样外，还须明示具体的采样方式。

【4】德国收紧防疫措施，措施将横跨圣诞及新年假期

德国总理默克尔与全国各邦首长举行会议，会后宣布自16日收紧防疫措施。措施横跨圣诞及新年假期，暂定至下月10日结束。默克尔指出，预期圣诞购物会令社交接触显著增加，引发病毒进一步扩散。她强调，联邦政府有责任避免公共卫生系统负荷过重，因此有必要紧急行动。新一轮措施生效期间，所有学校将要停课，学生以遥距方式学习；出售非必要商品的商铺及髪型屋暂停营业，而餐厅全面禁止堂食；雇主应该关闭办公室，容许雇员在家工作。另外，室内聚会的参加者最多为五人、而且必须来自不多于两个家庭，相关限聚令于平安夜至拆礼物日（12月24日至26日）会稍作放宽，期间一个家庭可以邀请不多于四名亲友于室内聚会。德国于周日新增逾两万宗“2019 冠状病毒”确诊病例，再多321名患者死亡，令累积确诊病例突破132万宗、死亡个案增至21787宗。

【5】新加坡政府批准使用“辉瑞 / BioNTech 疫苗”

新加坡政府昨日（14日）批准使用“辉瑞 / BioNTech 疫苗”，成为首个批准使用该款疫苗的东南亚国家。新加坡总理李显龙发表演说，指所有公民及长期居留人士均可免费接种疫苗，医护人员、长者等将获安排优先接种，而他与内阁其他成员亦会提前接种。他提到若一切按计划进行，明年第三季将有足够疫苗供全国人口接种。新加坡是首个批准使用“辉瑞 / BioNTech 疫苗”的东南亚国家，位于西亚的巴林、沙特阿拉伯及科威特此前已经先后批准使用该款疫苗。除了辉瑞，新加坡此前已向美国莫德纳及中国科兴支付疫苗订金。

【7】民调：多数日本民众反对明年举办东京奥运会

由于日本正遭受新一波冠病疫情的打击，大多数日本民众反对明年举办东京奥运会，他们希望进一步推迟或彻底取消这一大型活动。日本广播协会（NHK）周二发布的民调显示，仅有27%的受访者支持明年举办奥运会，而32%的人赞成取消奥运会，31%的受访者赞成进一步推迟奥运会。其余受访者表示不确定或并未给出答案。不过奥运会组织者和日本官员已经排除奥运会再次延期的可能性。东京都知事小池百合子受访时承认，目前大多数日本公众反对明年举办奥运会，但她认为“没有任何情况”会取消2020年东京奥运会，并坚信可以消除这些担忧。

【8】美国FDA肯定莫德纳疫苗安全有效

美国药监局（FDA）周二发布报告，经审查数据，认定麻省药厂莫德纳（Moderna）研发的2019冠状病毒疫苗有效，且无严重安全问题。此前研发数据显示，该疫苗有效性达94%。《华盛顿邮报》援引一位要求匿名的知情人士指，FDA最快将于本周五批准莫德纳疫苗上市。FDA报告指，莫德纳疫苗对所有年龄、族裔、性别人士均有良好的预防效果；对预防重症感染者尤其有效，临床试验中出现30例重症感染者，全部来自注射了安慰剂的对照组。FDA还指，莫德纳疫苗副作用现象普遍，包括疲劳、头痛、肌肉酸痛、注射部位疼痛等，通常为短期症状。

【9】英格兰南部发现病毒新变种 或致疫情迅速扩散

英国卫生部长汉考克透露，主要集中在英格兰南部的冠病病毒新变种，可能是造成该地区疫情迅速扩散的根源。 汉考克周一在国会说，至少有60个地区发生病毒变种，至今已有逾千起相关病例。初步分析显示，变异后的病毒传播速度较快，但影响疫苗有效性的可能性非常小。

【10】世卫专家组计划下个月访华 希望前往武汉调查

世卫组织专家小组正与中国专家讨论，希望下个月启程前往中国，并且可能会到最先通报出现冠病病毒疫情的武汉，但详细行程仍待确定。据《卫报》报道，专家小组成员之一的德国疾控机构科赫研究所生物学家伦德茨（Fabian Leendertz）强调，此次任务并非要找出有罪责的国家。他自出，从动物溢出到人类的事件一直发生，即使在德国也是如此，所以这次要在最大范围内找出发生了什么事，了解病毒怎样从动物跳到中间宿主，然后再传到人类，并重组这个传播过程，为未来可能再爆发疫情，减低风险做好准备。目前入境中国旅客都要隔离检疫两星期，伦德茨表示，不知专家组会否亦受到检疫限制，预计整个调查行程需时四至五星期，之后会撰写报告。但他相信，报告不会得到完整答案，可能有需要进一步做研究。参与过追踪西非伊波拉病毒的伦德茨表示，此次任务难度更高，因为确诊者的症状同流感类似，甚至完全没有病症。

【11】阿拉斯加州一医疗工作者接种辉瑞疫苗后出现严重过敏反应

《纽约时报》报道称，阿拉斯加州一名医疗工作者在接种辉瑞与BioNTech的冠病疫苗后，出现严重过敏反应，其目前住院接受治疗但情况稳定。《纽时》报道，这名医疗工作者在周二接种疫苗，辉瑞公司证实已同当局调查这起事件。英国有两名医疗工作者也有类似的过敏反应，促使英国政府告诉民众，如果有严重过敏史，就避免接种疫苗。美国监管机构在发布紧急批准时也同时警告民众，如果对疫苗内成分有已知过敏反应者，应避免接种。辉瑞发言人说：“我们还没有得到来自阿拉斯加关于潜在严重过敏反应报告的所有细节，但我们正在积极与当地卫生当局合作，以进行评估。”“我们将密切检测疫苗接种后出现严重过敏反应的所有报告，并在需要时更新标签语言。”

至于另外两个网络，相信与俄罗斯“互联网研究机构”（IRA）的过往活动有关。“互联网研究机构”被指由俄罗斯政府幕后操控，参与干预美国2016年总统大选，机构被俗称为“Trolls from Olgino（华语世界俗称：俄罗斯网络水军）”。Facebook 指出，这两个网络散播对中非共和国现任总统图瓦德拉（Faustin-Archange Touadéra）有利的虚假消息，目的是帮助这位亲俄总统争取连任，以推动俄罗斯在当地的商业及外交利益。Facebook 引述社交媒体分析公司的调查，指两个网络均会假扮成当地人发布伪造图片。除了中非共和国，两个网络还会伪装成来自利比亚、苏丹、阿尔及利亚、喀麦隆等另外十三个非洲国家的用户。此次是 Facebook 首次发现来自两个国家的阵营，为争夺在第三国的影响力而在平台上直接较量。同时，此次是 Facebook 首次点名指控与西方国家政府有联系的人士，但 Facebook 强调目前没有证据显示相关行动是由法国政府或军方指使。
