新闻思考

最近韩国少女在微博上求助的事件让大家知道大量韩国男性利用Deep fake技术将女性照片与不雅图像合成，并在网络上广泛传播。参与者不仅分享合成的图片，还附上受害者的详细个人信息，导致这些女性面临羞辱、骚扰甚至更严重的性侵害威胁。尽管事态严重，韩国政府在打击此类犯罪方面行动迟缓，许多受害者因此求助无门，只能在国际社会寻求帮助。直到最近几天公众的强烈抗议，韩国警方一直以 Telegram 的海外服务器为由，认为调查假裸照案件很困难。

韩国总统尹锡烈在内阁会议上指示当局彻底调查和解决日益严重的网络性犯罪问题，特别是利用Deep fake技术制作和传播露骨的色情图像。尹总统指出，最近社交媒体上迅速传播的深度伪造视频中，受害者往往是未成年人，而犯罪者多为青少年。

deepfake作为人工智能技术的一种应用，可以用来制作虚假的视频、音频等，从而欺骗观众。在这个信息爆炸的时代，我们如何判断信息的真实性，如何保护自己的隐私，如何保护未成年人的权益，这些都是我们需要思考的问题。

技术本身并没有对错，关键在于我们如何使用它。我觉得我们需要加强对Deep fake技术的监管，以及对于施暴者的惩罚，同时也需要加强对于受害者的保护。

在我国，针对deepfake的法律规定主要涉及肖像权、互联网法规以及个人信息等领域。首先，《民法典》第1019条规定：“任何组织或者个人不得以丑化、污损，或者利用信息技术手段伪造等方式侵害他人的肖像权。未经肖像权人同意，不得制作、使用、公开肖像权人的肖像，但是法律另有规定的除外。”根据该规定，即使没有营利目的和主观恶意，未经本人同意的深度伪造行为仍可能侵害肖像权。2023年4月发布的《生成式人工智能服务管理办法（征求意见稿）》，定义的“深度合成技术”包括人脸生成、人脸替换、人物属性编辑、人脸操控、姿态操控等生成或者编辑图像、视频内容中生物特征的技术。因deepfake涉及人脸等生物识别信息的显著编辑功能，deepfake服务提供者和技术支持者应当提示服务使用者依法告知人脸被编辑的个人（即肖像权人），并取得其单独同意。此外，为避免deepfake可能导致公众混淆或者误认，服务提供者应当在生成或者编辑的信息内容的合理位置、区域进行显著标识，向公众提示深度合成情况。

在美国，2019年，加利福尼亚州颁布了打击“深度伪造”的立法（Assembly Bill 602, AB602, California Civil Code Section 1708.86.），旨在打击在加利福尼亚州未经授权的deepfake色情视频（使用数字或电子技术描绘个人的露骨材料），以救济非自愿色情制品的受害者，使得其有权寻求禁令救济并得到补偿性和惩罚性损害赔偿及律师费。
