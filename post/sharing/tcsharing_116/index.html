本周摄影

新闻思考

这周是AI技术大爆发的一周。

微软和google分别发布了将ai技术集成到自己的office套件和谷歌文档中的消息，让大家担心自己的工作会被取代。目前这两套系统都只在几个公司内侧，还不知道效果如何。我对ai在office的表现并不觉得会完全取代人的工作，而是会大大提升我们的效率，让我们更专注于思考和创造。 google还发布了PaLM API 和 MakerSuite 可以使所有开发人员能够开始用生成性人工智能构建下一代应用程序，从而继续使人工智能变得容易使用。但目前也是内测阶段，不知道效果如何。

百度文心一言的发布让人们非常激动，这是一个我国的多模态模型，能够与人对话互动，回答问题，协助创作，高效便捷地帮助人们获取信息、知识和灵感。不过发布之后还是遭到了一些嘲讽，觉得和chatgpt的差距还是太远了，我们还是需要更加努力的去发展国内ai技术。

GPT4的发布让大家非常兴奋，这是一个多模态模型，可以接受图像作为输入并回答相关的问题。这个模型的出现，让人们对于ai的发展有了更多的期待。目前体验需要开通chatgpt-plus服务才能使用。另外一个变化就是，gpt4的技术文章其实一点细节都没有讲，只是简单的介绍了一下模型的结构，剩下的全是在讲模型的测试效果，让人觉得已经背离了openai的初衷，变成了closeai。在未来各个公司可能都不会公开自己的ai 模型了，一方面是训练需要大量的资源，直接开源会让自己蒙受损失，另一方面是训练出来的模型也是公司的核心竞争力，可以垄断市场。在之前，每个公司的基本架构都差不多，重点是怎么让用户来用自己的产品。现在ai 模型的差别直接决定了用户体验，不开源也可以理解。

现在，互联网都是免费的提供服务，大公司通过广告赚钱。在ai时代我觉得这种模式可能会走入历史，大公司会通过订阅费赚钱。因为在对话中很难插入很多广告，我们只有在使用这些ai服务的时候需要支付每月$10-$20才能享受效率的提升。

目前已知的收费：

  • chatgpt-plus $20/month
  • midjourney $10/month
  • notion ai $10/month
  • github copilot $10/month
  • …

在openai ceo Sam Altman 的设想中，以后的创业公司会基于这些大模型来创造各个领域所对应的ai应用，思考如何用ai创造更多的价值将是我们之后的最重要的事。

科技新闻

【1】新版ChatGPT火速发布

美国人工智能研究实验室OpenAI周二为其爆红聊天机器人ChatGPT发布了最新GPT-4语言模型，这距离ChatGPT的上线仅仅过去了4个月时间。

【#GPT4已在微软新必应上运行#】#GPT4发布OpenAI付款系统被挤爆#3月15日，微软副总裁兼消费者首席营销官Yusuf Mehdi发文确认新必应搜索引擎正在运行GPT-4。此前，OpenAI发布了多模态预训练大模型GPT-4，据介绍，GPT-4的回答准确性不仅大幅提高，还具备更高水平的识图能力，且能够生成歌词、创意文本，实现风格变化。此外，GPT-4的文字输入限制也提升至2.5万字，且对于英语以外的语种支持有更多优化。

OpenAI发布了最新的GPT-4，简单总结几个值得关注的点：

GPT-4是多模态模型（multimodal model）： 也就是它现在可以接受图像作为输入并回答相关的问题。

能输入的文本更长： GPT-4现在可以处理超过2.5万个单词的输入，基本上你能把一个中篇小说直接灌给它。

逻辑能力提升很大： 像GPT3.5在模拟律师考试中排名在倒数10%，而GPT-4的得分变成了正数前10%，其他学科的得分相应也有很大提升。

总之，GPT-4比前代的进步很大。

不过如果只是随意的交流，可能看不出太大的区别。但当问题的难度和复杂度上升的时候，GPT-4的优势就会展现出来。

所以还是得考验大家提问题的能力。

能在答案里直接生成图片、语音（带方言的），甚至视频。这要能根据描述直接生成视频那短视频营销号可有福了。。。不过李彦宏自己也说了现场演示的都是提前录好的，等发布会后拿到邀请码我想赶紧试试，但生成视频的没开放。。。

【3】炉石传说被移出杭州亚运会项目 根据杭州亚运会项目设置程序要求，亚奥理事会于2021年11月确定杭州亚运会电子竞技共设8个比赛项目。电子竞技属新兴体育项目，其办赛要求有别于传统体育项目。近年来，组委会在各方密切配合与支持下，严格按相关要求，不断优化电子竞技项目办赛所需条件。

《炉石传说》作为美国暴雪娱乐公司制作、在中国境内由中国网易公司代理运营的电竞项目，因双方商务合作到期，未就续约事项达成一致，导致该项目运营团队及服务器处于终止状态，已不具备列入杭州亚运会电子竞技项目的前置条件。

3月16日，经亚奥理事会第五次协调委员会会议审议决定取消《炉石传说》项目设置，杭州亚运会电子竞技将设7个比赛项目。（杭州2022年亚运会）

【5】2000年，微软以5861.97亿美元市值名列世界第一，英特尔以2770.96亿美元市值名列第六，那时候微软和英特尔的组合称之为wintel。

20多年过去了，微软还是名列世界市值最高的公司之一。3月17日的收盘价，微软市值为2.08万亿美元，而英特尔的市值是多少呢？1233.24亿美元，2000年市值的一半还不到。

2000年名不经传的英伟达，市值仅为48亿美元，以最近的收盘价计算，市值是6354.07亿美元。

2000年，英特尔约等于58个英伟达，但是在2023年，英特尔大约是英伟达的五分之一。

【6】谷歌眼镜彻底告别舞台！企业版全面停售，9月将结束软件支持

当地时间3月15日，谷歌宣布已停止销售Glass Enterprise智能眼镜，也将于今年9月起停止对该智能眼镜的软件支持。这是大型科技公司最早推出、也是最受认可的智能眼镜系列之一。2013年谷歌首次以 1500 美元推出谷歌眼镜，2017 年决定将产品定位为企业工具。

【7】Meta宣布第二轮大规模裁员 1万名员工将被裁退

3月14日，Facebook母公司、元宇宙龙头Meta宣布，为应对经济低迷，作为削减成本并提高工作效率计划的一部分，公司将再次裁员10000名员工。在此之前，Meta已于去年11月宣布了第一轮大规模裁员，约有11000多名员工受影响，约其员工总数的约13%。此外，Meta CEO马克•扎克伯格表示，公司还计划关闭额外5000个尚未填补的空缺职位。
