思考

在发了一年半的新闻的后，我开始思考一个问题，我们为什么需要看新闻？如果我不知道世界上发生大部分事情，生活会有什么不同吗？当然不会。

阅读新闻本质是对信息的处理，新闻本身只是事实，每个人都可以对其解读。如何解读新闻才是每个人与众不同的地方。之前我只是放新闻本身，希望不会影响各位自己的观点。我以后会多放放自己对于新闻的观点而不是仅仅分享新闻本身，所以将之前的国内国际新闻版块都拿掉。

同时分享的侧重也将科技和编程，科技新闻可以让我们发现前沿信息，编程项目可以让我们发现程序能做的事情。同时我也将新增life hack版块用来记录编程可以让生活更好的小技巧。

Life Hack

这次介绍的是scriptable，这是一款可以在iOS上自己定义小组件的软件

我们可以利用js进行编程，自己创作小插件。这里是他的文档，可以实现各种功能。

const lang = "en" //use the language of your choice "en", "de", "it", "fr", etc.
const url = `https://${lang}.wikipedia.org/api/rest_v1/page/random/summary`
const req = new Request(url)
const res = await req.loadJSON()
const i = new Request(res.thumbnail.source);
const img = await i.loadImage();

let widget = createWidget(res.title, img, res.content_urls.mobile.page)
if (config.runsInWidget) {
  // create and show widget
  Script.setWidget(widget)
  Script.complete()
}
else {
  widget.presentSmall()
}

function createWidget(title, img, widgeturl) {
  console.log("Title: "+title)
  let w = new ListWidget()
  w.backgroundColor = new Color("#1A1A1A")
  let image = w.addImage(img);
  image.centerAlignImage();
  let titleTxt = w.addText(title)
  titleTxt.textColor = Color.white()
  titleTxt.font = Font.systemFont(12)
  titleTxt.centerAlignText()
  w.url = widgeturl
  w.setPadding(0, 5, 0, 0)
  return w
}

这一段简单的代码可以实现一个随机wiki的小插件。

如果想在自己的iPhone上显示自己想追踪的信息，想办法获取到一个可以返回json数据的api就很容易将其显示在iOS上了。
